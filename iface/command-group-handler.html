Invoking kernels¶

sycl::handler¶

class handler;

The handler defines the interface to invoke kernels by submitting commands to a queue.

A handler can only be constructed by the SYCL runtime and is passed as an argument to the command group function. The command group function is an argument to submit.

See also

SYCL Specification Section 4.10.4

          sycl::access::target tgt_src, sycl::access::placeholder isPlaceholder_src,

          sycl::access::placeholder isPlaceholder_dest>

          sycl::accessor<T_dest, dim_dest, mode_dest, tgt_dest, isPlaceholder_dest> dest);

Copies memory from src to dest.

copy invokes the operation on a sycl::device. The source, destination, or both source and destination are Accessors. Source or destination can be a pointer or a shared_ptr.

mode_src

mode_dest

Mode for destination accessor

tgt_src

Target for source accessor

tgt_dest

Target for destination accessor

isPlaceholder_src

isPlaceholder_dest

Placeholder value for destination accessor

Parameters

src

source of copy

dest

destination of copy
